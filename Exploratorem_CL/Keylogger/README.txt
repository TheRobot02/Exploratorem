Version 1.0
In deze versie werkt de keylogger met unicode in plaats van ASCII of virtual keys.
Hoewel virtual keys nogsteed worden gebruikt is het maar een onderdeel om de juiste key input te vinden.

Er is een soort error handling in het begin van de main(). Het is nog de vraag of dit nog anders geimplementeerd kan worden.

Voor test redenen word de output zowel naar de terminal als naar een file gescheven om uit te zoeken of de data met de hulp van sockets.
Het huidige plan is om de data via mail te versturen. hooewel de data via een socket live naar een server te sturen ook een leuk project lijkt.
